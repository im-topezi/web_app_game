# web_app_game
The goal is to make a website that functions as a small text based RPG game.
Goals for the game:
Player can start a new game or continue one. Starting a new game randomly generates a new world from predefined chuncks that the player can interact with through forms. World also has NPC's and items that the player can interact with that are also randomly generated.
Player has stats that they can choose at the start of the game. Enemies also have stats. Stats can be raised by items obtained.
